26.在Python语言中，不属于Python标准库的是（）

​ A turtle

​ B time

​ C random

​ D PIL

答案

  1. C time属于内置的时间库
  2. B random 是随机库
  3. D pygame用于游戏开发；scipy用于网络爬虫；pdfminer用于处理文件
  4. B Flask 短小精悍的web开发框架
  5. D PIL处理图像的第三方库
  6. A numpy用于数据处理，包含矩阵运算
  7. D turtle海龟画图库；numpy用于数据处理；python即语言
  8. B 用于安装第三方库用pip
  9. A scrapy用于网络爬虫。
  10. B time属于时间库。
  11. A panda 用于数据分析
  12. C 网络爬虫库 scrapy
  13. A Beautifulsoup4用于解析和处理HTML和XML
  14. B -F 含义生成独立的打包文件
  15. C 不属于pip工具的只有C项
  16. A Matplotilb用于可视化 Flask用于web开发；PyMongo用于存储数据
  17. B 安装一个第三方库的命名格式 pip install <第三方库名>
  18. A 可以将图片等文件资源一并打包；CD项都属于Pyinstaller功能项
  19. D TImeSide用于web开源音频处理框架
  20. B request属于网络请求库
  21. A random库采用梅森旋转算法 并不是真正的随机数
  22. C -F 打包生成一个独立的可执行文件
  23. C redis-py 用于数据领域存储
  24. C random.getrandbites(k)生成一个k比特长度的随机整数
  25. A pandas用于数据分析 当然能用于金融数据分析
  26. D PIL属于第三方库，用于图片识别。
